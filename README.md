# Dsa-practice-questions.
A collection of C++ solutions to Data Structures and Algorithms (DSA) problems, organized by topic to strengthen problem-solving and coding skills.

## Topics Covered

- Arrays
- Strings
- Recursion
- Sorting
- Searching
- Linked Lists
- Stacks
- Queues
- Trees
- Binary Search Trees
- Graphs
- Hashing
- Dynamic Programming
- Greedy Algorithms
- Backtracking
- Bit Manipulation
